Simple Server
=============

An exceedingly simple server that listens for incoming connections, prints
their contents and responds `hello`.

Runs on port 8080.

Usage
-----

1. Clone
2. Cargo Run

Contributing
------------

will always welcome the work of others. Please make sure you check the Code
